- Overview Tailwind works by scanning your project for utility classes, then generating all of the necessary CSS based on the classes you've actually used.
- This makes sure your CSS is as small as possible, and is also what makes features like arbitrary values possible.
- How classes are detected Tailwind treats all of your source files as plain text, and doesn't attempt to actually parse your files as code in any way.
